# Army Fight via threads & via processes

## About project 
We have two army. The numbers of soldiers of each army increases in random size.

Write projects that perform own soldiers count addition and enemy soldiers count subtraction.
Write project that creates two child processes from father process that will perform that task.
Write project that creates threads that will perform that task.
